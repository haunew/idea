# 学术引用格式参考

## APA 7th Edition（社会科学常用）

### 期刊文章
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, volume(issue), pages. https://doi.org/xx.xxx/yyyy
```

### 书籍
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle (Edition). Publisher.
```

### 在线资源
```
Author, A. A. (Year, Month Day). Title of page. Site Name. URL
```

## GB/T 7714-2015（中国国家标准）

### 期刊文章
```
[序号] 主要责任者. 文献题名[J]. 刊名, 年, 卷(期): 起止页码.
```
示例：
```
[1] 张三, 李四. 人工智能在教育中的应用研究[J]. 教育研究, 2023, 44(5): 45-52.
```

### 专著
```
[序号] 主要责任者. 书名[M]. 版本项. 出版地: 出版者, 出版年: 引文页码.
```

### 学位论文
```
[序号] 主要责任者. 题名[D]. 保存地: 保存单位, 年份.
```

### 会议论文
```
[序号] 主要责任者. 题名[C]//会议名称. 出版地: 出版者, 年份: 页码.
```

### 电子文献
```
[序号] 主要责任者. 题名[EB/OL]. (发布日期)[引用日期]. 网址.
```

## MLA 9th Edition（人文科学常用）

### 期刊文章
```
Author(s). "Title of Article." Title of Journal, vol. #, no. #, Year, pp. xx-xx.
```

### 书籍
```
Author. Title of Book. Publisher, Year.
```

## Chicago 17th Edition

### 脚注/尾注格式
```
1. Firstname Lastname, Title of Book (Place: Publisher, Year), page.
```

### 参考文献格式
```
Lastname, Firstname. Title of Book. Place: Publisher, Year.
```

## IEEE（工程技术常用）

```
[序号] A. Author and B. Author, "Title of article," Title of Journal, vol. x, no. x, pp. xxx-xxx, Month Year.
```

## 引用类型标识符（GB/T 7714）

| 标识 | 含义 |
|------|------|
| M | 专著（Monograph） |
| C | 会议录（Conference） |
| N | 报纸文章（Newspaper） |
| J | 期刊文章（Journal） |
| D | 学位论文（Dissertation） |
| R | 报告（Report） |
| S | 标准（Standard） |
| P | 专利（Patent） |
| EB/OL | 电子公告/在线（Electronic Bulletin/Online） |
