## Directory Structure

```

├── README.md              # Chinese version
├── README-EN.md           # This file (English)
```

## Workflow

```

```

```yaml

```

## Quick Start

1. **Have an idea**: Create a file in `ideas/`, capture quickly
2. **Want to dig deeper**: Develop in `research/`, build framework
3. **After discussion**: Update status, organize conclusions
4. **Ready for output**: Use `academic-paper-writing` skill for formal paper
