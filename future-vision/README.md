## 快速开始 / Quick Start

1. **有灵感时 / Have an idea**: 在 `ideas/` 创建新文件，快速记录 / Create a file in `ideas/`, record quickly
2. **想深入时 / Want to dig deeper**: 在 `research/` 展开论述，构建框架 / Develop in `research/`, build framework
3. **讨论后 / After discussion**: 更新状态，整理结论，形成论文大纲 / Update status, organize conclusions
4. **成体系时 / Ready for output**: 使用 `academic-paper-writing` skill 输出正式论文 / Use skill for academic paper
