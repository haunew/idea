# 学术论文写作

本 Skill 提供学术论文写作的全流程指导和支持。

## 核心工作流程

### 1. 论文规划阶段

- 明确论文类型（期刊论文/学位论文/会议论文/综述）
- 确定目标期刊/会议的格式要求
- 制定论文大纲（IMRaD 结构或其他适用结构）
- 规划各章节篇幅和写作时间表

### 2. 文献综述

- 系统检索相关文献
- 整理文献笔记和引用信息
- 识别研究空白（research gap）
- 构建理论框架

### 3. 论文撰写

标准学术论文结构（IMRaD）：

1. **标题（Title）**：简洁、准确、有信息量
2. **摘要（Abstract）**：背景、方法、结果、结论，200-300字
3. **引言（Introduction）**：研究背景、文献综述、研究问题、研究意义
4. **方法（Methods）**：研究设计、数据收集、分析方法，确保可重复
5. **结果（Results）**：客观呈现数据，图表清晰
6. **讨论（Discussion）**：解释结果、与文献对比、局限性、未来方向
7. **结论（Conclusion）**：总结主要发现、理论贡献、实践意义
8. **参考文献（References）**：格式统一、引用准确

### 4. 学术规范检查

- **引用格式**：根据期刊要求使用 APA/MLA/Chicago/GB/T 7714 等格式
- **学术诚信**：避免抄袭，正确引用和改写
- **语言规范**：学术英语/中文表达，避免口语化
- **图表规范**：清晰标注、编号、来源说明

## 文件格式支持

### 常用格式

| 格式 | 用途 | 工具 |
|------|------|------|
| **.docx** | 最常用，期刊投稿、学位论文 | Microsoft Word / WPS / LibreOffice |
| **.tex** | 理工科常用，公式排版优秀 | Overleaf / TeX Live / MiKTeX |
| **.pdf** | 最终提交格式，不可编辑 | 所有系统通用 |
| **.md** | 初稿写作，版本控制友好 | Typora / VS Code |

### 推荐工作流

1. **初稿阶段**：Markdown / Word 快速写作
2. **修改阶段**：Word（导师审阅、批注修改）
3. **排版阶段**：LaTeX（公式多）或 Word（人文社科）
4. **投稿阶段**：PDF（最终提交）

### Word 文档处理

Word 是学术论文最常用的格式。处理 .docx 文件：

```python
# 安装依赖
pip install python-docx

# 读取/修改 Word 文档
from docx import Document
doc = Document('paper.docx')

# 统计字数
total = sum(len(p.text) for p in doc.paragraphs)
print(f"总字符数: {total}")
```

详见 [references/docx-guide.md](references/docx-guide.md) 完整指南。

## 论文讨论与完善

深度讨论是论文质量的关键。使用 [references/paper-discussion-tips.md](references/paper-discussion-tips.md) 中的框架：

### 5个关键问题

1. **技术路径优先级** — 近期/中期/远期如何选择？
2. **核心论点锐度** — 温和演进 vs 范式革命？
3. **反方观点回应** — 如何主动承认限制？
4. **杀手级场景** — 哪个应用最能打动读者？
5. **论文定位** — 综述/观点/技术/预测？

### 常见陷阱

- ❌ 专业术语堆砌 → ✅ 每个术语都有通俗解释
- ❌ 过度承诺 → ✅ 使用条件句和时间框架
- ❌ 忽视限制 → ✅ 主动设置Limitations章节
- ❌ 缺乏对比 → ✅ 现在vs未来的对比表格

## 写作技巧

### 标题撰写
- 准确概括研究内容
- 避免缩写和 jargon
- 控制在 15-20 词以内

### 摘要撰写
- 背景（1-2句）
- 方法（1-2句）
- 主要结果（2-3句）
- 结论/意义（1-2句）

### 引言结构（漏斗式）
1. 宏观背景 → 2. 领域现状 → 3. 存在问题 → 4. 研究目的 → 5. 研究贡献

### 方法部分要点
- 足够详细以便他人重复
- 使用过去时态
- 说明伦理审批（如涉及人体/动物研究）
- 软件和统计方法需注明版本

### 结果与讨论
- **结果**：只报告事实，不解释
- **讨论**：解释意义，与文献对比，承认局限

## 常见期刊类型及要求

| 类型 | 特点 | 结构重点 |
|------|------|----------|
| 实证研究 | 原创数据 | IMRaD |
| 综述 | 系统总结 | 引言-文献综述-讨论-结论 |
| 方法论文 | 新方法/工具 | 方法细节、验证 |
| 案例研究 | 具体实例 | 背景-案例-分析-启示 |
| 理论论文 | 概念框架 | 文献-理论构建-讨论 |

## 中文学术论文特殊要求

- 遵循 GB/T 7714-2015 引用格式
- 注意中英文标点规范
- 图表标题需中英文双语
- 基金项目、作者简介格式
