# Stage 3: 区域无线供电技术研究

## 摘要

区域无线供电（Region-Scale Wireless Power）是能源演进的第三阶段，通过在城市/区域范围内部署能量基站，实现类似移动通信的广域覆盖。本文分析了微波输能（MPT）和毫米波技术、基站网络架构、动态功率管理，以及城市级部署策略。研究表明，区域级无线供电将在2035-2045年实现商用，彻底改变城市能源基础设施。

---

## 1. 技术背景与需求分析

### 1.1 Stage 2的局限

**覆盖范围限制**：
- 单栋建筑内无线
- 出门后无覆盖
- 街道上、车辆中仍需传统能源
- 跨建筑移动时能量中断

**用户体验痛点**：
```
场景：从家到公司
家里：建筑无线供电 ✓
电梯：无覆盖 ✗
街道：无覆盖，手机用电池 ✗
地铁：无覆盖 ✗
公司：另一栋建筑，重新连接 ✗

结果：体验碎片化，仍需充电宝
```

### 1.2 区域级需求

| 需求 | 指标 | 说明 |
|------|------|------|
| **覆盖范围** | 1-10km半径 | 城市/社区级别 |
| **移动支持** | 支持高速移动 | 车辆、行人 |
| **功率等级** | 10W-10kW | 手机到电动汽车 |
| **并发用户** | 10万+ | 城市人口密度 |
| **可靠性** | 99.99% | 电信级可靠性 |

## 2. 技术原理：微波与毫米波

### 2.1 微波输能（MPT）

**技术原理**：
```
发射天线 ──[微波 2.45GHz/5.8GHz]──▶ 整流天线(Rectenna)
    │                                      │
  相控阵                                二极管整流
  (波束成形)                            (DC输出)
```

**关键参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| **频率** | 2.45 GHz / 5.8 GHz | ISM频段，全球可用 |
| **波长** | 12.2 cm / 5.2 cm | 适合天线尺寸 |
| **效率** | 50-70% | 发射到接收 |
| **距离** | 数公里 | 视距传输 |
| **功率** | kW-MW级 | 基站级别 |

**优势**：
- 远距离传输（公里级）
- 高功率（支持电动汽车）
- 定向传输（安全性高）

**挑战**：
- 需要视距（不能穿建筑物）
- 雨雾衰减（毫米波尤甚）
- 生物安全（功率密度限制）

### 2.2 毫米波输能

**技术参数**：

| 参数 | 数值 | 说明 |
|------|------|------|
| **频率** | 28 GHz / 60 GHz | 5G毫米波频段 |
| **波长** | 10.7 mm / 5 mm | 高方向性 |
| **效率** | 30-50% | 较高频段损耗大 |
| **距离** | 100m-1km | 较短 |
| **功率** | 100W-1kW | 适合小范围 |

**优势**：
- 极高方向性（精准定位）
- 天线尺寸小（便于部署）
- 与5G基础设施共享

**挑战**：
- 雨雾衰减严重
- 距离短
- 需要更多基站

### 2.3 混合方案

```
MPT（2.45GHz）：广覆盖，高功率，远距离
    +
毫米波（28GHz）：精准定位，高带宽，短距离
    +
RF（2.4GHz）：背景供能，低功率，全向
    =
区域全覆盖
```

## 3. 系统架构设计

### 3.1 整体架构

```
┌─────────────────────────────────────────┐
│           区域无线供电网络               │
│                                          │
│   电网 ──▶ 区域能量中心（MW级）          │
│                │                         │
│       ┌────────┴────────┐                │
│       ↓                 ↓                │
│   MPT基站群         毫米波基站群         │
│   (广覆盖)          (精准覆盖)           │
│       │                 │                │
│       └────────┬────────┘                │
│                ↓                         │
│         中继/反射网络                    │
│                ↓                         │
│    ┌───────────┼───────────┐             │
│    ↓           ↓           ↓             │
│   手机       车辆       建筑             │
│  (10W)     (10kW)     (100kW)            │
└─────────────────────────────────────────┘
```

### 3.2 区域能量中心

**功能**：
- 接收电网电力（MW级）
- 转换为微波/毫米波
- 协调多个基站
- 智能调度功率

**组成**：

| 组件 | 功能 | 规模 |
|------|------|------|
| **主控制器** | 网络协调 | 电信级 |
| **MPT发射机** | 生成微波 | 1-10MW |
| **毫米波阵列** | 生成毫米波 | 100-1000单元 |
| **储能系统** | 削峰填谷 | MWh级 |
| **冷却系统** | 散热 | 工业级 |

**选址**：
- 城市边缘（减少人口密集区辐射）
- 高地（视线覆盖好）
- 电网接入方便

### 3.3 MPT基站

**天线设计**：
```
相控阵天线：
- 尺寸：10m × 10m
- 单元数：10,000个
- 波束：可电子转向
- 增益：40-50 dBi

波束特性：
- 主瓣宽度：1-2度（非常窄）
- 旁瓣抑制：>20dB
- 扫描范围：±60度
```

**覆盖计算**：
```
基站高度：100m（塔或高楼）
覆盖半径：10km（视距）
覆盖面积：314 km²

城市基站数：
- 北京：10个基站全覆盖
- 上海：8个基站全覆盖
- 深圳：5个基站全覆盖
```

### 3.4 毫米波基站

**设计**：
```
与5G基站共址：
- 共享天线塔
- 共享电源
- 共享回传
- 共享维护

天线阵列：
- 256单元
- 波束宽度：10度
- 覆盖：100-500m
- 功率：1kW
```

**密度**：
```
城市：每平方公里10-20个
郊区：每平方公里2-5个
农村：每平方公里0.5-1个
```

### 3.5 中继网络

**为什么需要中继**：
- MPT需要视距
- 城市建筑阻挡
- 需要绕过障碍物

**中继方式**：

| 方式 | 原理 | 应用 |
|------|------|------|
| **空中中继** | 无人机/飞艇 | 临时覆盖、灾害恢复 |
| **地面中继** | 反射板、中继塔 | 固定盲区覆盖 |
| **建筑中继** | 建筑顶部接收-转发 | 街道覆盖 |

## 4. 动态功率管理

### 4.1 波束成形与跟踪

**相控阵原理**：
```
10,000个天线单元
    ↓
每个单元独立控制相位
    ↓
相位叠加形成波束
    ↓
波束指向任意方向
    ↓
无需机械转动，电子扫描
```

**设备跟踪**：
```
定位系统：
- GPS：粗略定位（米级）
- 5G定位：精确定位（分米级）
- 雷达：实时跟踪（厘米级）

跟踪算法：
1. 获取设备位置
2. 计算波束指向
3. 调整相控阵相位
4. 实时跟踪移动
5. 预测移动轨迹

响应时间：<1ms（支持高速移动）
```

### 4.2 功率自适应

**距离补偿**：
```
自由空间路径损耗：
L = 20log₁₀(d) + 20log₁₀(f) + 32.45

其中：
d = 距离（km）
f = 频率（MHz）

2.45GHz，10km：
L = 20log₁₀(10) + 20log₁₀(2450) + 32.45
  = 20 + 67.8 + 32.45
  = 120.25 dB

功率补偿：
- 近距离（100m）：发射功率1W
- 远距离（10km）：发射功率100kW
```

**动态调整**：
```
设备接近基站：降低功率（节省能量，减少干扰）
设备远离基站：增加功率（维持接收）
设备静止：稳定功率
设备高速移动：预测性调整
```

### 4.3 多用户调度

**时分多址（TDMA）**：
```
时间轴分割：
t1: 给设备A充电
t2: 给设备B充电
t3: 给设备C充电
...

周期：1ms（支持1000用户/秒）
```

**空分多址（SDMA）**：
```
多波束同时：
波束1 → 设备A（方向1）
波束2 → 设备B（方向2）
波束3 → 设备C（方向3）
...

同时服务多个用户
```

**功率分配算法**：
```
优先级：
1. 紧急服务：救护车、消防车
2. 关键基础设施：医院、通信
3. 付费用户：高等级服务
4. 普通用户：基础服务

公平性：
- 确保最低服务（维持通信）
- 按需分配额外功率
```

## 5. 安全性设计

### 5.1 生物安全防护

**功率密度限制**：

| 标准 | 限值 | 设计目标 |
|------|------|---------|
| **FCC** | 10 mW/cm² | <1 mW/cm² |
| **ICNIRP** | 50 W/m² | <5 W/m² |
| **中国** | 40 μW/cm² | <4 μW/cm² |

**多层防护**：

```
第一层：功率控制
- 最大发射功率限制
- 根据距离自动调整
- 避免近距离高功率

第二层：波束管理
- 窄波束（减少扩散）
- 精确指向（不照射人体）
- 快速扫描（减少暴露时间）

第三层：禁区设置
- 人口密集区降低功率
- 学校、医院特殊保护
- 无人机监控人群

第四层：应急停止
- 检测到异常立即断电
- 人工紧急停止按钮
- 自动故障保护
```

### 5.2 电磁兼容

**与通信系统共存**：
```
频段分离：
- WiFi/蓝牙：2.4GHz（避开MPT 2.45GHz）
- 5G：3.5GHz/28GHz（与毫米波分开）
- 卫星：Ku/Ka波段（无冲突）

协调机制：
- 时分：MPT发射时通信暂停
- 频分：不同频段同时工作
- 空分：不同方向互不干扰
```

## 9. 结论

区域无线供电（Stage 3）是连接建筑级和全球级的关键：

1. **技术突破**：MPT和毫米波使远距离无线供电成为可能
2. **体验革命**：城市内完全无线，像移动通信一样自然
3. **基础设施**：需要大规模投资，但成本可控
4. **演进基础**：为太空太阳能积累技术和经验

预计2035-2045年，区域无线供电将从试点走向规模部署，成为城市基础设施的重要组成部分。
